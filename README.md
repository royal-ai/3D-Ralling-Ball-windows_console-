# 3D-Ralling-Ball-windows_console-
Cancel changes## 游戏简介
3D-Ralling-Ball 是一款基于Windows控制台的3D字符画小游戏，没有借助其他的图形库。
游戏的玩法是通过w,a,s,d操控小球左右闪躲，跳跃来躲避障碍物，获取星星来获取得分。
